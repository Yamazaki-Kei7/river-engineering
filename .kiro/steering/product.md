# プロダクト概要

hyetograph-cli は、河川工学・都市排水設計・洪水解析向けにハイエトグラフ（降雨時間分布図）を生成するRust製コマンドラインツール。既存のExcel VBAマクロ（RKEISANサブルーチン）の計算ロジックをRustで再実装している。

## コア機能

- **降雨強度計算**: Sherman型降雨強度式 `K = C / ((T * I)^A + B)` による増分雨量の算出
- **雨量分布配置**: 3パターンの時間配置（前方集中型、交互ブロック法による中央集中型、後方集中型）
- **マルチフォーマット出力**: PNG棒グラフ（plotters）および/またはCSVデータファイルの生成

## 対象ユースケース

- 河川技術者が水文解析のための設計降雨時間分布を作成する
- 技術報告書・プレゼン資料用のハイエトグラフチャートを生成する
- 手作業のExcel VBAワークフローを再現可能なCLIツールに置き換える

## 価値提案

- **VBA互換**: 元のExcel VBAマクロと数値的に同一の結果を出力（IEEE 754 f64精度）
- **再現性**: CLIベースのワークフローによりスクリプト化・自動化が可能
- **クロスプラットフォーム**: 外部ランタイム依存なしのPure Rustバイナリ
